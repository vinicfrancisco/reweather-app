## Running App

To run this app you need to follow this steps:

- Installing dependences

```
  yarn
```

- Installing iOS dependences (only if using iOS)

```
  cd ios && pod install && cd ..
```

- Running app

```
  yarn ios
```
  or
```
  yarn android
```
